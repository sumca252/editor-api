# Editor API 
![Node CI](https://github.com/sumca252/editor-api/actions/workflows/node.js.yml/badge.svg?branch=main)

Backend API for [editor](https://github.com/sumca252/editor). 

## Install dependencies
```
npm install
```
## Start the server 
```
npm start
```

## Eslint 
```
npm run eslint
```
## Fix eslint errors 
```
npm run fix
```
